# Alternative landing pages

60 standalone HTML landing-page experiments for **Derivative Duel**.

## Contents

- `pages/variant-01.html` … `variant-60.html` — individual landings
- `index.html` — gallery of all variants
- `manifest.json` — id / title / fonts / motion metadata
- `screenshots/` — desktop + mobile PNG captures + `REPORT.md`
- `generate-variants.mjs` — regenerates all pages
- `screenshot-all.mjs` — local static server + Chrome screenshots

## Commands

```bash
cd alt
npm install
npm run generate
npm test
npm run screenshots
```

Open `index.html` or `screenshots/report.html` in a browser.
